
# Install

## Run install script

```bash
curl -fsSL https://raw.githubusercontent.com/her0e1c1/dotfiles/master/install.sh | bash
```

## Run make install

```bash
make install
```

## Setup VSCODE

Sign in with github account by selecting `Backup and Sync Settings`
