# HHKB Cheatsheet

Use this when you forget the Bluetooth pairing flow for HHKB.

## Pair New Device

1. Press `Fn + Q` to enter pairing standby mode.
2. Press one of `Fn + Ctrl + 1..4` to choose the device slot to store.
3. Open Bluetooth settings on the device you want to connect, then select HHKB.

## Switch Device

1. Press `Fn + Ctrl + 1..4` to switch to the target device slot.
2. Press `Fn + Ctrl + 0` to switch back to the USB connection.

## Remove One Pairing

1. Press `Fn + Q` to enter pairing standby mode.
2. Press one of `Fn + Ctrl + BS + 1..4` to delete the pairing stored in that slot.

## Remove All Pairings

1. Press `Fn + Q` to enter pairing standby mode.
2. Press `Fn + Z + BS` to delete all saved pairing information.

## Cancel Pairing Mode

1. Press `Fn + X` while HHKB is in pairing standby mode.
2. Confirm that HHKB has returned to normal input mode.

## Notes

- HHKB can store up to four Bluetooth pairings.
