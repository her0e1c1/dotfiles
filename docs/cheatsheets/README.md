# Command Cheatsheets

Quick index for the commands and keybindings that are easy to forget in this dotfiles setup.

## Tools

| Page | Focus |
| --- | --- |
| [nvim.md](nvim.md) | Neovim commands and this config's keymaps |
| [tmux.md](tmux.md) | Custom prefix, pane/window/session operations |
| [tig.md](tig.md) | Custom tig bindings, git/gh flows, destructive actions |
| [shell-cli.md](shell-cli.md) | Shell aliases, fzf helpers, readline bindings, container helpers |

## Usage

- Start here when you forget a key or alias.
- Prefer each tool page for day-to-day use; keep this file as the entrypoint.
- When you change `.tmux.conf`, `.tigrc`, `.profile`, or `neovim/lua`, update the matching page in the same commit.
